# Языковой тренажёр

## Суть проекта:
* пользователь выбирает язык и тему;
* программа показывает случайные слова, цель пользователя – правильно их перевести;
* программа проверяет ответы и считает очки;
* есть возможность запросить подсказку;
* слова разделены по темам.


## Структура проекта:
```
language_simulator/
language_trainer/
│
├── main.py                     # Точка входа 
│
├── requirements.txt

│   └── languages.json          # Список языков
```
